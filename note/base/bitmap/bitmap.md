## Bitmap API

### 与、或、非、异或、与非操作

```c
bitmap_and			//与
bitmap_or			//或
bitmap_complement	//非
bitmap_xor			//异或
bitmap_andnot		//与非
```

### 置位、清零、填充

```c
bitmap_set		//置位操作
bitmap_clear	//清零操作
bitmap_zero		//清零操作
bitmap_fill 	//填充操作
```
